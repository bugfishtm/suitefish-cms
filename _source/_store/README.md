# 📁 Store

✅ Folder included in Backups  
✅ Modified files in this folder will not be overwritten by updates/upgrades.  
❌ Do not manually modify or add/edit files in this folder.  

The _store folder manages cache files from downloaded items via the internal store, as well as files for deploying modules, core versions, or software through the CMS. Core updates do not affect this folder.

**Sub folders:**

- _core: Folder for deployed Core Releases
- _module: Folder for deployed Module Releases
- _software: Folder for deployed Software Releases

This folder stores deployed store files, including core releases, module releases, and software releases. The backend will create the required folder structure upon first initialization.

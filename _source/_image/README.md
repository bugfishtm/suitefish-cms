# 📁 Image Modules

✅ Folder included in Backups  
✅ Modified files in this folder will not be overwritten by updates/upgrades.  
⚠️ Developers may manually edit folder content or deploy modules manually.  

Contains installed and active image modules. It's recommended to manage these through the Administrator Interface; manual edits are not advised. Refer to the documentation for module development. 

This folder stores deployed image modules that are currently active. The `disabled` folders stores inactive modules. Modules placed there are recognized as inactive and handled accordingly based on their type.

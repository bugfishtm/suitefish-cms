# 🛠️ Developers

This folder contains resources for developers working with **Suitefish CMS**, including examples and tutorials to help you build, extend, and customize modules.

- **Site Modules** (`_site`, `_sm`) – Standalone website modules.
- **Extension Modules** (`_extension`, `_ext`) – Add-on functionality for other modules.
- **Image Modules** (`_image`, `_img`) – Image-based deployments and external CMS integrations.
- **Software Modules** (`_software`, `_sw`) – Applications for deployment to Suitefish users.
- **Docker Modules** (`_docker`, `_dkr`) – Manage containers via the server manager.
- **Theme Modules** (`_theme`, `_th`) – Customize the admin interface with theming.

# 📁 Module Skeletons  

### 📌 Site Modules (_site or _sm)  
Standalone website modules that function independently.  

### 🔧 Extension Modules (_extension or _ext)  
Enhance website modules with additional functionality.  

### 🖼️ Image Modules (_image or _img)  
Deployable image-based modules, including unrelated CMS projects like WordPress.  

### 💻 Software Modules (_software or _sw)  
For software developers deploying applications to customers.  

### 🐳 Docker Modules (_docker or _dkr)  
Modules for managing Docker containers via our server manager extension.  

### 🎨 Theme Modules (_theme or _th)  
Customize the Administrator Module with theming enhancements.

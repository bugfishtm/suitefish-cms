# Core-System - Function Library

Information about included php core functions.

---

## lib.init.worker.php

| **Function Name**       | **Parameters**                                                                                                                                                                                                                         | **Usage / Purpose**                                                                 |
|--------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| `hive__worker`          | - **$object**: contains the DB connection (expects `$object["mysql"]`) <br> - **$hive_mode**: the “mode” or module name for the worker job <br> - **$script_name**: the script/job name <br> - **$extension_mod** *(optional)*: extension module name <br> - **$parameters** *(optional)*: can be array (converted to JSON) or JSON string <br> - **$custom_data** *(optional)*: custom metadata, can be array (serialized) <br> - **$script_c_type**: script type (`php`, `py`, etc., required) | Core function that validates input, prepares job data, and inserts a new worker task/job into `_TABLE_WORKER_`. Returns inserted job ID. |
| `hive__worker_php`      | Same as `hive__worker` **except `script_c_type` is fixed as `"php"`**                                                                                                                          | Shortcut for creating **PHP script jobs** without typing `"php"` manually.            |
| `hive__worker_sh`       | Same as `hive__worker` **except `script_c_type` is fixed as `"sh"`**                                                                                                                           | Shortcut for creating **Shell script jobs** (sh).                                     |
| `hive__worker_bash`     | Same as `hive__worker` **except `script_c_type` is fixed as `"bash"`**                                                                                                                         | Shortcut for creating **Bash script jobs**.                                           |
| `hive__worker_python`   | Same as `hive__worker` **except `script_c_type` is fixed as `"py"`**                                                                                                                           | Shortcut for creating **Python script jobs**.                                         |
| `hive__worker_perl`     | Same as `hive__worker` **except `script_c_type` is fixed as `"pl"`**                                                                                                                           | Shortcut for creating **Perl script jobs**.                                           |

## lib.init.user.php

### Get Functions

| **Function Name**       | **Parameters**                                                                                                                                                      | **Usage / Purpose**                                                                                                   |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| `hive__user_get`         | - **$object**: expects DB connection inside (`$object["mysql"]`) <br> - **$user_id**: numeric user ID                                                           | Fetches and returns **all user data** from `_TABLE_USER_` where `id = $user_id`. Returns `false` if input invalid.     |
| `hive__user_lang_get`    | - **$object** <br> - **$user_id** <br> - **$mode**                                                                                                               | Retrieves the user’s **language setting** for a specific mode from `hive_extradata`.                                   |
| `hive__user_theme_get`   | - **$object** <br> - **$user_id** <br> - **$mode**                                                                                                               | Retrieves the user’s **theme setting** from `hive_extradata`.                                                           |
| `hive__user_theme_sub_get` | - **$object** <br> - **$user_id** <br> - **$mode**                                                                                                             | Retrieves the user’s **sub-theme** setting.                                                                            |
| `hive__user_color_get`   | - **$object** <br> - **$user_id** <br> - **$mode**                                                                                                               | Retrieves the user’s **color scheme** setting.                                                                         |
| `hive__user_key_get`     | - **$object** <br> - **$user_id** <br> - **$mode** <br> - **$key**                                                                                                | Generic getter: retrieves **any custom key/value** stored under a mode in `hive_extradata`.                            |

### Setup Functions

| **Function Name**       | **Parameters**                                                                                                                                                            | **Usage / Purpose**                                                                                       |
|--------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| `hive__user_lang_set`   | - **$object** <br> - **$user_id**: numeric <br> - **$mode** <br> - **$theme_name**: must be in `_HIVE_LANG_ARRAY_`                                                        | Sets/updates the user’s **language preference** inside `hive_extradata`. Validates language against a whitelist. |
| `hive__user_theme_set`  | - **$object** <br> - **$user_id** <br> - **$mode** <br> - **$theme_name**: must be in `_HIVE_THEME_ARRAY_`                                                              | Sets/updates the user’s **main theme selection**. Validates against allowed themes.                        |
| `hive__user_theme_sub_set` | - **$object** <br> - **$user_id** <br> - **$mode** <br> - **$theme_name**                                                                                             | Sets/updates the user’s **sub-theme** value.                                                              |
| `hive__user_color_set`  | - **$object** <br> - **$user_id** <br> - **$mode** <br> - **$theme_name**                                                                                                | Sets/updates the user’s **color scheme**.                                                                |
| `hive__user_key_set`    | - **$object** <br> - **$user_id** <br> - **$mode** <br> - **$key**: string key name <br> - **$value**: value to set                                                       | **Generic setter**: stores arbitrary metadata under a given `mode` and `key` in the serialized `hive_extradata`. |

## lib.init.trigger.php


| **Function Name**    | **Parameters**                                                                 | **Usage / Purpose**                                                                                                               |
|-----------------------|---------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| `hive__trigger`       | - **$object**: application context <br> - **$trigger_name**: name of trigger    | Executes **all triggers** with the given name: first from each site module in `_HIVE_MODE_ARRAY_`, then from current site’s extensions if `_HIVE_MODE_` is defined. |
| `hive__trigger_site`  | - **$object** <br> - **$trigger_name**                                          | Executes **only site module triggers** (in `_HIVE_MODE_ARRAY_`), ignoring extensions.                                             |
| `hive__trigger_ext`   | - **$object** <br> - **$trigger_name**                                          | Executes **only extension triggers** for the currently defined `_HIVE_MODE_`, ignoring site modules.                              |

## lib.init.inject.php

| **Function Name**     | **Parameters**                                                    | **Usage / Purpose**                                                                                                  |
|------------------------|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| `hive__inject`         | - **$object**: application context <br> - **$inject**: inject name | Executes **both site module injects** and **extension injects** for the given inject name (`inject.X.php` if present). |
| `hive__inject_site`    | - **$object** <br> - **$inject**                                  | Executes only **site module injects** (from `_HIVE_MODE_ARRAY_` paths).                                                |
| `hive__inject_ext`     | - **$object** <br> - **$inject**                                  | Executes only **extension injects** (from `$object["extensions_path"]`) for the currently defined `_HIVE_MODE_`.       |

## lib.init.download.php

| **Function Name**       | **Parameters**                                                                | **Usage / Purpose**                                                                                                      |
|--------------------------|-------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| `hive__download_view`    | - **$filePath**: path to file <br> - **$filenamex** *(optional)*: override filename | Sends the file to browser. If the extension is *inline-safe* (jpg, png, pdf, mp4, etc.), it’s displayed inline, otherwise forced as download. Supports partial content (HTTP range). |
| `hive__download_force`   | - **$filePath** <br> - **$filenamex** *(optional)*                            | Always forces download of the file (never inline view). Supports range requests for resume/partial download.              |
| `hive__download_mimeTypes` | *(none)*                                                                     | Returns an array mapping file extensions → MIME types to determine correct **Content-Type** headers.                      |

## lib.init.library.php

| **Function Name**               | **Parameters**                                                                                                           | **Usage / Purpose**                                                                                                                         |
|---------------------------------|--------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| `hive__init_site_header`        | - **$object** <br> - **$site_name**                                                                                      | Prepares and returns a configuration array for a given **site module** (path, name, prefix, cookie, language handler, etc.).                  |
| `hive__init_extension_header`   | - **$object** <br> - **$extension_name** <br> - **$sitemod_name** *(optional)*                                           | Prepares and returns a configuration array for a given **extension** (path, name, prefix, cookie, language handler, etc.).                   |
| `hive__extension_path`          | - **$hive_mode**                                                                                                         | Returns all directories under `/_data/{hive_mode}/_extension` (the available extensions for that mode).                                       |
| `hive__require_once`            | - **$object** <br> - **$filepath**                                                                                       | Wrapper for `require_once()`; includes a PHP file only once, returns `true`.                                                                 |
| `hive__require`                 | - **$object** <br> - **$filepath**                                                                                       | Wrapper for `require()`; includes a PHP file, returns `true`.                                                                                 |
| `hive__require_x`               | - **$filepath**                                                                                                          | Includes a file **only if it contains `version.php`**; returns `$x` variable (if defined inside) or `false`.                                  |
| `hive__trim`                    | - **$string**                                                                                                            | Safe wrapper for `trim()`, works when string can be `null`.                                                                                  |
| `hive__hsc`                     | - **$string**                                                                                                            | Wrapper for `htmlspecialchars()` with `ENT_QUOTES`, safely escaping HTML entities.                                                            |
| `hive__hen`                     | - **$string**                                                                                                            | Wrapper for `htmlentities()` with `ENT_QUOTES`, encoding HTML entities.                                                                       |
| `hive__access`                  | - **$object** <br> - **$rights_local** (string or array) <br> - **$global_search** *(optional)* <br> - **$inital_override** *(optional)* | Checks whether **logged-in user** has specific permissions. Supports local permission check or global search (user and group permissions).     |
| `hive__url`                     | - **$array**: URL parameters <br> - **$following_parameters** *(bool)*                                                   | Builds relative URL based on array input. Supports both **SEO-friendly URLs** or traditional query-string format depending on settings.       |
| `hive__is_main`                 | - **$object**                                                                                                            | Checks if the current HIVE site/module is the **main active instance** (used for multi-site installations, admin site detection).             |
| `hive__folder_create`           | - **$folderpath** <br> - **$forwardfile** *(bool)* <br> - **$denie_access** *(bool)* <br> - **$chmode** *(default 0770)* | Creates a folder if not present. Optionally adds `index.php` for forward protection and `.htaccess` restrictions if requested.                 |
| `hive__file_tail`               | - **$filename** <br> - **$lines** *(default: 1000)*                                                                     | Reads a file from the end (like Unix `tail`), returning the last N lines efficiently.                                                         |
| `hive__in_array`                | - **$needle** <br> - **$haystack**                                                                                       | Recursive search in nested arrays. Returns `true` if the needle is found at any depth.                                                        |

## lib.init.template.php

| **Function Name**                 | **Parameters**                                                                                                                     | **Usage / Purpose**                                                                                                       |
|-----------------------------------|------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| `hive__template_mail`              | - **$object** <br> - **$template** (template name or identifier) <br> - **$mail** (recipient email) <br> - **$user_id** (user reference) <br> - **$url_if_needed** (inject URL, e.g., action link) | Sends a mail based on a template. Pulls user details (first/last name), applies template substitutions, and sends via `$object["mail"]`. Supports built-in defaults for register, recover, and mail-change templates. |
| `hive__template_mail_activate`     | - **$object** <br> - **$get_token** *(default `"ref_token"`)* <br> - **$get_user** *(default `"ref_user"`)* <br> - **$message** *(bool, display messages?)* <br> - **$redirect** *(optional redirect URL)* | Handles **mail change confirmation** when user clicks a link in their email. Confirms via `user->mail_edit_confirm()`, shows results/messages, and optionally redirects. |
| `hive__template_user_activate`     | - **$object** <br> - **$get_token** *(default `"ref_token"`)* <br> - **$get_user** *(default `"ref_user"`)* <br> - **$message** *(bool)* <br> - **$redirect** *(optional)* | Handles **new user account activation** (after registration). Calls `user->activation_confirm()`, reports result messages, optional redirect. |
| `hive__template_login`             | - **$object** <br> - **$cookies_allow** *(bool: allow “remember me”)* <br> - **$message** *(bool: show messages)*                   | Handles **default login flow**: checks IP ban, CSRF tokens, credentials. Executes `user->login_request(...)`, shows messages depending on success/error code. Supports persistent login cookies. |
| `hive__template_recover_request`   | - **$object** <br> - **$rec_url** *(optional recovery URL)* <br> - **$get_token** *(default `"rec_token"`)* <br> - **$get_user** *(default `"rec_user"`)* | Handles **password recovery flow (request step)**. Accepts email, checks CSRF, calls `user->recover_request()`. If valid, auto-sends recovery email via `hive__template_mail(...)`. Returns codes on success/error. |


## lib.init.default.php


| **Function Name**              | **Parameters**                                                                                                                                                                                                                               | **Usage / Purpose**                                                                                                    |
|--------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| `hive__default_volt_footer`    | - **$object** *(optional, default: false)* <br> - **$footer** *(string, default: "")*: footer HTML/text to output <br> - **$classes** *(default: `"bg-white rounded shadow p-5 mb-3 mt-3"`)*: CSS classes for footer <br> - **$end_div** *(bool, default: true)* | Renders the default **Volt footer**: optionally closes an open `<div>`, outputs a styled `<footer>` if content provided, then closes `<main>`, `<body>`, and `<html>`. |
| `hive__default_volt_header`    | - **$object**: main object context <br> - **$tabtitle** *(string, default "")*: browser tab title <br> - **$metaextensions** *(string)*: extra meta or link tags <br> - **$theme_default** *(string, default `"dark"`)* <br> - **$mainclass** *(string)*: CSS class for `<body>` <br> - **$defaultclasses** *(bool, default true)*: include default Volt JS/CSS libraries? <br> - **$iconimage** *(string, optional)*: favicon/icon (base64 or URL) <br> - **$excludecore** *(bool, default true)* | Renders the default **Volt HTML header** and opening `<body>`. Outputs meta tags, favicon, sets document `<title>`. Optionally includes default Volt CSS/JS libraries and extra extensions. Manages theme (light/dark) from session or user preferences. |

## lib.init.error.php

| **Function Name** | **Parameters**                                                                                                                                                                                                                   | **Usage / Purpose**                                                                                           |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| `hive__error`     | - **$title** *(string)*: Main error title (required) <br> - **$subtitle** *(string/false, default false)*: Optional subtitle <br> - **$description** *(string/false, default false)*: Longer error description <br> - **$exit** *(bool, default true)*: Whether to `exit()` after outputting error page <br> - **$code** *(int/false)*: Optional HTTP error code (e.g. `404`, `500`) <br> - **$backtohome** *(bool, default false)*: Show “Back to Home” link <br> - **$icon** *(string, default `"danger"`)*: Error icon style (`danger`, `info`, `warning`, `cog`) | Outputs a **full animated HTML error page** with styled message, subtitle, optional description, SVG icon, and back-to-home link if enabled. Sets HTTP response code if provided. Exits after rendering by default. |

## 🛠️ Installation

You have two ways installing this module.

1. Login to your suitefish instance and browse our official store on your page. If you find the module you are looking for just download the package. Then navigate to the Extension Package Area and enable the downloaded extension.

2. Navigate to the Extension Package Area inside the suitefish instance. Go to the upload section and upload the modules .zip file.  Then navigate to the Extension Package Area and enable the downloaded extension.

## 📄 Documentation 

If you are a developer you can find examples of modules in the _developers folder at the suitefish-cms github repository if you want to create an own module! For more information about the Suitefish CMS: https://github.com/bugfishtm/suitefish-cms

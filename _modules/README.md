📁 **Example Modules**

This folder houses modules for development, testing, or use, split into three sections:

**Site Modules:** Full website modules capable of acting standalone, you can find examples in the _site folder.

**Extension Modules:** Modules applicable to website modules, enhancing their functionality, you can find examples in the _extension folder.

**Image Modules:** Image modules to be deployed, this can be full other website projects not related to this cms like wordpress and misc, you can find examples in the _image folder.

**Software Modules:** If you are a software developer, you can use this modules to deploy software to customers, you can find examples in the _software folder.

Feel free to explore and utilize these modules to enhance your projects!

📁 **Example Modules**

This folder houses modules for development, testing, or use, split into three sections:

**Site Modules:** Full website modules capable of acting standalone. (_site or files starting with _sm)

**Extension Modules:** Modules applicable to website modules, enhancing their functionality. (_extension or files starting with _ext)

**Image Modules:** Image modules to be deployed, this can be full other website projects not related to this cms like wordpress and misc. (_image or files starting with _img)

**Software Modules:** If you are a software developer, you can use this modules to deploy software to customers. (_software or files starting with _sw)

**Docker Modules:** If you are a using our docker server manager extension, you can use this modules to deploy docker containers. (_docker or files starting with _dkr)

**Theme Modules:** Enhance the Theme of the Administrator Module with Theming Modules. (_theme or files starting with _th)

Feel free to explore and utilize these modules to enhance your projects!

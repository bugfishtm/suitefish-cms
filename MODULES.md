
# Module IDs 

Here you can see reserved module "rname" values and namespaces. Request your own module id and get in touch with out team. For more information see our official www.suitefish.com website.

| rname | description|
|------|---------|
| _* | All module rnames starting with _ are reserved for official modules. |
| _xd* | Reserved for debugging Site Modules. |
| _site | Site Module example Module. |
| _administrator | Reserved for Suitefish Site Module. |
| _smobr | Reserved for book renting Site Module. |
| _smbugfish | Reserved for bugfish Site Module. |

| _extplanehook | Plane Webhook Extension Module. |
